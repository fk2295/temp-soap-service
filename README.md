# Temp SOAP Service

A Python SOAP temperature conversion web service with WSDL support, built using Spyne.

## Features
- Fahrenheit to Celsius
- Celsius to Fahrenheit
- SOAP 1.1 support
- Auto-generated WSDL

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
